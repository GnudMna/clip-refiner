# ClipRefiner

```
使用方法:
  引数なし: システムトレイに常駐し、クリップボードを監視して自動変換
  --codec指定: クリップボードの内容を一度だけ変換

Usage: ClipRefiner.exe [OPTIONS]

Options:
  -c, --codec <CODEC>
          コーデックの指定

          [possible values: encode, decode]

  -h, --help
          Print help (see a summary with '-h')

  -V, --version
          Print version
```
